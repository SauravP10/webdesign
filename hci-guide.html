Human Computer Interaction & User Interfaces


What is HCI?

Human computer interaction is the study design, evaluation, and implementation of interactive computing systems for human use and the major phenomena surrounding them. Consist of three parts: the person, the computer, the way they work together

What is a User Interface?

User interface design refers to graphical user interfaces and other forms that designers build in software to create interfaces for computerized devices that users find easy to use and pleasurable.

Types of user interfaces include:

  • GUI (graphical user interface): users interact with visual representations on digital control panels
  • Voice controlled interfaces (VCIs): users interact with their voices
  • Gesture based interfaces: Users engage with 3D design spaces through bodily motion
